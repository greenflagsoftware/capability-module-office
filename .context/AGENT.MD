# Persistent Agent Memory

This folder (`.context`) contains memory files for maintaining context across agent sessions.

## Instructions

1. **Always read `CLAUDE.md` at the start of a new session.** This file provides project-level guidance, architecture, tool patterns, and layout for the repository.

2. **Always read `memory.md` at the start of a new session.** This file contains the persistent agent memory accumulated from previous sessions.

3. **`memory.md` is the canonical file for persisting agent memory.** Use it to record important context, decisions, project status, tasks in progress, and any other information that should survive across agent sessions. Update it as the project evolves.
